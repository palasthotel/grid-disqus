== Installation ==

1. Upload `grid-disqus-boxes.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `grid-disqus-boxes` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Put Disqus Boxes on your Grid Landingpages
